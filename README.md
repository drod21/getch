# getch

> Asynchronous data fetch library

[![NPM](https://img.shields.io/npm/v/getch.svg)](https://www.npmjs.com/package/getch) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save getch
```

## Usage

```tsx
import * as React from 'react'

import { useMyHook } from 'getch'

const Example = () => {
  const example = useMyHook()
  return (
    <div>
      {example}
    </div>
  )
}
```
